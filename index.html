1. Gate of Hell

자바 콘솔에서 동작하는 게임입니다.
왕국 지하에 있는 지옥문이 열리고
주인공은 지옥문에서 쏟아져 나오는 악마들을 막기 위해
직접 지옥문 안으로 들어가 싸워 나간다는 내용의 게임입니다.

사용된 스킬: JAVA







2. Bird House

반려조와 함께 생활하시는 분들을 위해서
반려조에게 필요한 용품들만
따로 판매를 하면 좋을 것 같아서 만든
반려조 전문 쇼핑몰 앱입니다.
버드하우스라는 가상의 반려조 전용 용품점에서
운영하는 앱입니다.

사용된 스킬: JAVA , Apache, PHP, MySQL, Firebase
